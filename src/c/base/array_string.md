## 一、数组
### 1、定义
c++ 支持数组数据结构，它可以存储一个固定大小的相同类型元素的顺序集合。数组是用来存储一系列数据，但它往往被认为是一系列相同类型的元素。

### 2、语法
```bash
type arrayName [arraySize]
```

### 3、初始化数组
```bash
double balance[5] = { 1000.0, 2.0, 3.4, 7.0, 50.0 };
```

### 4、访问数组中的元素
```bash
double salary = balance[1];
```
