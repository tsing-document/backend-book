### 055-就业管理系统

**技术框架：** Spring Boot + 若依框架

#### **主要功能模块**

1. **系统管理模块**
    - 用户管理
    - 角色管理
    - 菜单管理
    - 部门管理
    - 岗位管理
    - 字典管理
    - 参数设置
    - 通知公告

2. **系统工具模块**
    - 表单工具
    - 代码生成
    - 系统接口

3. **企业管理模块**
    - 企业信息管理
    - 招聘信息管理
    - 投递简历记录

4. **学生信息管理模块**
    - 学生信息管理
    - 学生个人简历
    - 统计就业学生
    - 协议书管理
