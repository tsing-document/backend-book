## 一、if 语句

if 语句主要用来控制程序的走向。

## 二、简单的示例
```python
# 输入

# name == 'xiaoming' 单个判断
# 多个判断使用 and 拼接，这个是 and 两边的条件都满足才可以。
# 多个判断使用 or 拼接，这个是 or 两边条件有一边满足就行。
# 判断某个元素是否存在集合中 in。
# 判断某个元素是否不存在集合中 not in。

names = ['xiaoming','xiaoli', 'xialiu']
for name in names:
    if name == 'xiaoming':
        print('my name is '  + name)
    else:
        print(name)

# 输出
my name is xiaoming
xiaoli
xialiu

# 第二种格式
if temp_var >= 2:
    print()
elseif:
    print()
else:
    print()
```
